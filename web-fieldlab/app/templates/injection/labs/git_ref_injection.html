git ref 拼接命令注入

vuln：cmd = f"git --version && echo using-ref={ref}" + shell=True
safe：ref 必须匹配 ^[A-Za-z0-9._/-]{1,64}$，且只 subprocess.run(["git", "--version"])
PoC：?ref=HEAD;id   ?ref=$(id)
（容器无 git 时 vuln 仍会用 sh 模拟拼接执行，PoC 仍可观测）
