教学观察点

  • 接口层的输入从 form-urlencoded 变成 JSON，不改变 SQL 拼接本质。
  • 推荐老师同时展示浏览器 fetch 和 curl，帮助学生把 API 注入与页面注入统一起来。
  • 若要讲自动化测试，可基于这个接口再扩展最小 PoC 脚本。
